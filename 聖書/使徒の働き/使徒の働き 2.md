#使徒の働き

五旬節の日になって、皆が同じ場所に集まっていた。
すると天から突然、激しい風が吹いて来たような響きが起こり、彼らが座っていた家全体に響き渡った。
また、炎のような舌が分かれて現れ、一人ひとりの上にとどまった。
すると皆が聖霊に満たされ、御霊が語らせるままに、他国のいろいろなことばで話し始めた。
さて、エルサレムには、敬虔なユダヤ人たちが、天下のあらゆる国々から来て住んでいたが、
この物音がしたため、大勢の人々が集まって来た。彼らは、それぞれ自分の国のことばで弟子たちが話すのを聞いて、呆気にとられてしまった。
彼らは驚き、不思議に思って言った。「見なさい。話しているこの人たちはみな、ガリラヤの人ではないか。
それなのに、私たちそれぞれが生まれた国のことばで話を聞くとは、いったいどうしたことか。
私たちは、パルティア人、メディア人、エラム人、またメソポタミア、ユダヤ、カパドキア、ポントスとアジア、
フリュギアとパンフィリア、エジプト、クレネに近いリビア地方などに住む者、また滞在中のローマ人で、
ユダヤ人もいれば改宗者もいる。またクレタ人とアラビア人もいる。それなのに、あの人たちが、私たちのことばで神の大きなみわざを語るのを聞くとは。」
人々はみな驚き当惑して、「いったい、これはどうしたことか」と言い合った。
だが、「彼らは新しいぶどう酒に酔っているのだ」と言って、嘲る者たちもいた。
ペテロは十一人とともに立って、声を張り上げ、人々に語りかけた。「ユダヤの皆さん、ならびにエルサレムに住むすべての皆さん、あなたがたにこのことを知っていただきたい。私のことばに耳を傾けていただきたい。
今は朝の九時ですから、この人たちは、あなたがたが思っているように酔っているのではありません。
これは、預言者ヨエルによって語られたことです。
『神は言われる。
終わりの日に、わたしは
すべての人にわたしの霊を注ぐ。
あなたがたの息子や娘は預言し、
青年は幻を見、老人は夢を見る。
その日わたしは、
わたしのしもべにも、はしためにも、
わたしの霊を注ぐ。
すると彼らは預言する。
また、わたしは上は天に不思議を、
下は地にしるしを現れさせる。
それは血と火と立ち上る煙。
主の大いなる輝かしい日が来る前に、
太陽は闇に、月は血に変わる。
しかし、主の御名を呼び求める者は
みな救われる。』
イスラエルの皆さん、これらのことばを聞いてください。神はナザレ人イエスによって、あなたがたの間で力あるわざと不思議としるしを行い、それによって、あなたがたにこの方を証しされました。それは、あなたがた自身がご承知のことです。
神が定めた計画と神の予知によって引き渡されたこのイエスを、あなたがたは律法を持たない人々の手によって十字架につけて殺したのです。
しかし神は、イエスを死の苦しみから解き放って、よみがえらせました。この方が死につながれていることなど、あり得なかったからです。
ダビデは、この方について次のように言っています。
『私はいつも、主を前にしています。
主が私の右におられるので、
私は揺るがされることはありません。
それゆえ、私の心は喜び、
私の舌は喜びにあふれます。
私の身も、望みの中に住まいます。
あなたは、私のたましいをよみに捨て置かず、
あなたにある敬虔な者に
滅びをお見せにならないからです。
あなたは私に、
いのちの道を知らせてくださいます。
あなたの御前で、
私を喜びで満たしてくださいます。』
兄弟たち。父祖ダビデについては、あなたがたに確信をもって言うことができます。彼は死んで葬られ、その墓は今日に至るまで私たちの間にあります。
彼は預言者でしたから、自分の子孫の一人を自分の王座に就かせると、神が誓われたことを知っていました。
それで、後のことを予見し、キリストの復活について、『彼はよみに捨て置かれず、そのからだは朽ちて滅びることがない』と語ったのです。
このイエスを、神はよみがえらせました。私たちはみな、そのことの証人です。
ですから、神の右に上げられたイエスが、約束された聖霊を御父から受けて、今あなたがたが目にし、耳にしている聖霊を注いでくださったのです。
ダビデが天に上ったのではありません。彼自身こう言っています。
『主は、私の主に言われた。
あなたは、わたしの右の座に着いていなさい。
わたしがあなたの敵を
あなたの足台とするまで。』
ですから、イスラエルの全家は、このことをはっきりと知らなければなりません。神が今や主ともキリストともされたこのイエスを、あなたがたは十字架につけたのです。」
人々はこれを聞いて心を刺され、ペテロとほかの使徒たちに、「兄弟たち、私たちはどうしたらよいでしょうか」と言った。
そこで、ペテロは彼らに言った。「それぞれ罪を赦していただくために、悔い改めて、イエス・キリストの名によってバプテスマを受けなさい。そうすれば、賜物として聖霊を受けます。
この約束は、あなたがたに、あなたがたの子どもたちに、そして遠くにいるすべての人々に、すなわち、私たちの神である主が召される人ならだれにでも、与えられているのです。」
ペテロは、ほかにも多くのことばをもって証しをし、「この曲がった時代から救われなさい」と言って、彼らに勧めた。
彼のことばを受け入れた人々はバプテスマを受けた。その日、三千人ほどが仲間に加えられた。
彼らはいつも、使徒たちの教えを守り、交わりを持ち、パンを裂き、祈りをしていた。
すべての人に恐れが生じ、使徒たちによって多くの不思議としるしが行われていた。
信者となった人々はみな一つになって、一切の物を共有し、
財産や所有物を売っては、それぞれの必要に応じて、皆に分配していた。
そして、毎日心を一つにして宮に集まり、家々でパンを裂き、喜びと真心をもって食事をともにし、
神を賛美し、民全体から好意を持たれていた。主は毎日、救われる人々を加えて一つにしてくださった。
( 使徒の働き 2:1-47 JDB )

문법 설명
주어진 구절에는 다양한 문법적 요소들이 사용되었습니다. 몇 가지 주요 특징을 설명해 드리겠습니다.
 * 동사의 て形 (테형) 연결: 일본어 문장에서 자주 사용되는 형태로, 동사 뒤에 'て'를 붙여 여러 문장이나 구절을 연결하거나 부가적인 의미를 나타냅니다.
   * 集まっていた: '集まる(모이다)'의 て形 + 'いる(있다)'의 과거형 → 모여 있었다.
   * 座っていた: '座る(앉다)'의 て形 + 'いる(있다)'의 과거형 → 앉아 있었다.
   * 現れ: '現れる(나타나다)'의 て形 → 나타나서 (이어지는 동작을 연결)
   * 満たされ: '満たす(채우다)'의 수동형 '満たされる'의 て形 → 채워져서
   * 話すのを聞いて: '話す(말하다)' + 'の(것)' + '聞く(듣다)'의 て形 → 말하는 것을 듣고
   * 思って: '思う(생각하다)'의 て形 → 생각해서
   * 〜て言った: 〜라고 말했다.
   * 〜て殺したのです: 〜해서 죽인 것입니다.
 * 受身形 (수동태): '動詞の未然形 + れる/られる' 형태로, 주어가 어떤 행위의 대상이 될 때 사용됩니다.
   * 満たされ: '満たす(채우다)'의 수동태 '満たされる' (채워지다). '성령에 채워졌다'는 의미로 사용되었습니다.
   * 語らせる: '語る(말하다)'의 사역동사 '語らせる' (말하게 하다). '御霊が語らせるままに (성령이 말하게 하는 대로)'라는 표현으로 사용되었습니다.
   * 引き渡された: '引き渡す(넘기다)'의 수동태 '引き渡される' (넘겨지다).
   * 十字架につけて殺したのです: '十字架につける(십자가에 못 박다)' + '殺す(죽이다)'의 て形 연결.
   * 解き放たれて: '解き放つ(풀어주다)'의 수동태 '解き放たれる' (풀어지다).
   * 繋がれていることなど、あり得なかったから: '繋がる(묶이다)'의 수동태 + 'いる' → 묶여있는.
   * よみがえらせました: 'よみがえる(되살아나다)'의 사역동사 'よみがえらせる' (되살아나게 하다).
 * 引用 (인용): 남의 말이나 글을 인용할 때 사용됩니다.
   * 「〜」と言った。: 「〜」라고 말했다.
   * 『〜』: 성경 구절을 인용할 때 자주 사용되는 형태입니다.
 * 〜ままに: '〜대로', '〜하는 대로'라는 의미로, 어떤 상태나 행동에 따름을 나타냅니다.
   * 御霊が語らせるままに: 성령이 말하게 하는 대로.
 * 〜ではないか: '〜이 아닌가', '〜이지 않은가'라는 의미로, 확인이나 감탄, 상대방에게 동의를 구할 때 사용됩니다.
   * ガリラヤの人ではないか: 갈릴리 사람이 아닌가.
 * 〜とは、いったいどうしたことか: '〜라니, 도대체 어떻게 된 일인가'라는 의미로, 놀라움이나 의문을 강하게 표현할 때 사용됩니다.
   * 話を聞くとは、いったいどうしたことか: 말을 듣는다는 것이, 도대체 어떻게 된 일인가.
 * 〜まで: '〜까지'라는 의미로, 시간적, 공간적, 또는 정도의 한계를 나타냅니다.
   * 今日に至るまで: 오늘에 이르기까지.
   * 〜とするまで: 〜으로 삼을 때까지.
 * 〜はずがない: '〜할 리가 없다', '〜일리가 없다'라는 의미로, 강한 부정이나 확신을 나타냅니다.
   * あり得なかったから: 있을 수 없었기 때문이다.
 * 〜である/であるから: '〜이다/〜이기 때문이다'라는 의미로, 공식적이거나 단정적인 표현입니다. 문어체에서 자주 사용됩니다.
   * 預言者でしたから: 예언자였으므로.
   * 証人です: 증인입니다.
 * 〜ねばなりません/〜なければなりません: '〜해야만 한다'라는 의미로, 의무나 필요를 나타냅니다.
   * 知らなければなりません: 알아야만 합니다.
 * 目的語 + を + 動詞: 기본적인 목적어 표현입니다.
   * 神を賛美し: 신을 찬미하고.
단어 설명
구절에 등장하는 주요 단어들을 설명해 드리겠습니다.
 * 五旬節 (ごじゅんせつ): 오순절. 유대교의 주요 축제 중 하나로, 곡물 수확을 감사하고 시내 산에서 율법을 받은 것을 기념하는 날입니다. 기독교에서는 성령 강림을 기념하는 날이기도 합니다.
 * 響き (ひびき): 울림, 메아리. 여기서는 '소리'나 '울려 퍼지는 소리'를 의미합니다.
 * 響き渡った (ひびきわたった): 울려 퍼졌다. 소리가 넓게 퍼지는 것을 나타냅니다.
 * 炎のような舌 (ほのおのようなした): 불꽃 같은 혀. 성령 강림 시 나타난 현상을 묘사한 표현입니다.
 * とどまった (とどまった): 머물렀다, 멈췄다. 어떤 곳에 고정되거나 정착하는 것을 의미합니다.
 * 聖霊 (せいれい): 성령. 기독교에서 하나님(하느님)의 영 또는 그리스도의 영을 일컫는 말입니다.
 * 御霊 (みたま): 성령, 영. 성령을 높여 부르는 말입니다.
 * 語らせる (かたらせる): 말하게 하다. 사역 동사로, 어떤 주체가 다른 이로 하여금 말하게 하는 것을 의미합니다.
 * 他国 (たこく): 다른 나라, 이국.
 * ことば: 말, 언어.
 * 敬虔な (けいけんな): 경건한. 종교적인 믿음이 깊고 진실한 태도를 나타냅니다.
 * 天下の (てんかの): 천하의, 세상의 모든.
 * 物音 (ものおと): 소리, 물체가 내는 소리.
 * 呆気にとられてしまった (あっけにとられてしまった): 어리둥절해졌다, 얼이 빠졌다. 예상치 못한 일에 놀라 정신을 차리지 못하는 상태를 나타냅니다.
 * ガリラヤ: 갈릴리. 이스라엘 북부 지방으로, 예수님의 고향이기도 합니다.
 * パルティア人、メディア人、エラム人...: 구절에 언급된 다양한 지역과 민족들. 당시 로마 제국과 주변 지역의 사람들입니다.
 * 滞在中 (たいざいちゅう): 체류 중인.
 * 改宗者 (かいしゅうしゃ): 개종자. 다른 종교로 바꾼 사람을 의미합니다.
 * 神の大きなみわざ (かみのおおきなみわざ): 하나님의 크신 역사(행하심). 하나님의 위대한 행위나 능력을 의미합니다.
 * 当惑して (とうわくして): 당황해서, 어쩔 줄 몰라.
 * 嘲る (あざける): 조롱하다, 비웃다.
 * ぶどう酒 (ぶどうしゅ): 포도주.
 * 声を張り上げ (こえをはりあげ): 목소리를 높여. 큰 소리로 말하는 것을 의미합니다.
 * 耳を傾けていただきたい (みみをかたむけていただきたい): 귀 기울여 주십시오. 겸손하게 요청하는 표현입니다.
 * 預言者 (よげんしゃ): 예언자. 하나님의 말씀을 전하거나 미래를 예언하는 사람입니다.
 * 御霊 (みたま): 성령.
 * 注ぐ (そそぐ): 붓다, 쏟다. 여기서는 성령을 부어주시는 것을 의미합니다.
 * 幻 (まぼろし): 환상. 현실이 아닌 상상이나 꿈에 나타나는 것.
